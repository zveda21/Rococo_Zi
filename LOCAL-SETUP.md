# Local Setup Guide

- Prepare dependencies and build the project
```bash
chmod +x local-setup.sh
./local-setup.sh
```

- Open a new terminal and run:
```bash
./local-rococo-artist.sh
```

- Open a new terminal and run:
```bash
./local-rococo-gateway.sh
```

- Open a new terminal and run:
```bash
./local-rococo-userdata.sh
```

- Open a new terminal and run:
```
./local-client.sh
```
